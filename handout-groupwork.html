3Claude Code にプロジェクトを作らせる（5分）

やること

  1. Claude Code のデスクトップアプリを開く
  2. 新しいプロジェクトフォルダを作る（デスクトップに offkai-demo フォルダを作っておく）
  3. 下のプロンプトから、Step 1で選んだものを貼り付けて送る

最初のプロンプト（コピペ用）

Step 1で選んだものを1つ選び、○○ の部分を自分の情報に書き換えてから送ってください。書き換えが難しければ仮の情報でOKです。

A. ショップ紹介ページを作る場合

デスクトップの「offkai-demo」フォルダに移動して、
シンプルなショップ紹介ページを作ってください。

【私の情報】
- 名前: ○○ショップ（自分のショップ名に変えてください）
- 何を売っているか: ○○（例: ドライフラワーのリース）
- 価格帯: ○○円〜○○円
- 連絡先: （InstagramのURLなど）

要件:
- ファイルは index.html 1つだけ
- シンプルでかわいいデザイン（パステルカラー）
- スマートフォンでも見やすいレイアウト
- ブラウザでファイルをダブルクリックしたら開けるもの

B. 自己紹介 / ポートフォリオページを作る場合

デスクトップの「offkai-demo」フォルダに移動して、
シンプルな自己紹介 / ポートフォリオページを作ってください。

【私の情報】
- 名前: ○○（自分の名前や活動名に変えてください）
- やっていること: ○○（例: SNS運用サポート、資料作成、ハンドメイド販売）
- できること: ○○（例: 相談対応、デザイン、文章作成）
- 実績や経験: ○○（なければ「これから始めたいこと」でもOK）
- 連絡先: （メール、InstagramのURLなど）

要件:
- ファイルは index.html 1つだけ
- はじめて見る人にも何をしている人かわかる構成
- やさしく信頼感のあるデザイン
- スマートフォンでも見やすいレイアウト
- ブラウザでファイルをダブルクリックしたら開けるもの

C. イベント告知ページを作る場合

デスクトップの「offkai-demo」フォルダに移動して、
シンプルなイベント告知ページを作ってください。

【イベント情報】
- イベント名: ○○（例: 春のミニマルシェ）
- 日時: ○○年○月○日 ○:○○〜○:○○
- 場所: ○○（例: ○○カフェ、オンライン、地域センター）
- 内容: ○○（例: ハンドメイド作品の販売、体験レッスン）
- 参加方法: ○○（例: 予約不要、InstagramのDMで予約）
- 連絡先: （InstagramのURLなど）

要件:
- ファイルは index.html 1つだけ
- 日時・場所・参加方法がすぐわかる構成
- 楽しそうで見やすいデザイン
- スマートフォンでも見やすいレイアウト
- ブラウザでファイルをダブルクリックしたら開けるもの

4対話しながらページを育てる（15分）

Claude Code が index.html を作ったら、ブラウザで開いて確認します。次の「追加お願いリスト」から気になるものを試してみましょう。

追加お願いリスト（コピペ用）

カード形式の項目を追加したい場合

このページの内容に合うカードを3つ追加してください。
ショップ紹介ページなら「商品カード」、
自己紹介ページなら「できること・実績・相談メニュー」、
イベント告知ページなら「見どころ・当日の流れ・参加方法」にしてください。
それぞれに短い見出しと一言説明を入れてください。

色やデザインを変えたい場合

全体のカラーテーマを「ミントグリーン系」に変えてください。
フォントも丸みのある感じにしてください。

SNSリンクを追加したい場合

ページの下部に「Instagramを見る」ボタンを追加してください。
ボタンをクリックしたら https://instagram.com/（自分のURL）に飛ぶようにしてください。

気に入らない部分を直したい場合

ヘッダー部分のレイアウトが気になります。
タイトルをもっと大きく、背景色を白から薄いピンクに変えてください。
コツ: 「なんか違う」と感じたら「もっと○○にしてほしい」と伝えるだけでOK。完璧でなくていいです。

2「質問の型」を3要素でおさらい（5分）

AIへの質問は 3つの要素 を埋めれば9割うまくいきます。

① 役割

私は何をしている人か

例: 私は○○をしています。
② 状況

今、何に困っているか

例: 今、○○に困っています（○○がしたいです）。
③ 欲しい出力

どんな形式で返してほしいか

例: ○○という形式で出してください。
大切なこと: AIは自分の文脈を知らない。「あなたは何者で、何に困っているか」を毎回伝えるのがコツ。

3まなんちゃん版プロンプトを試す（実演）（5分）

次のプロンプトをそのまま Claude（または Claude Code）に送ります。

私はお花の寄植えと花苗をマルシェで販売している個人事業主です。
月に2〜3回、地元のマルシェに出店しています。

今、仕入れたお花の種類・価格と、当日の販売状況をExcelやGoogleスプレッドシートで管理したいのですが、
どう記録すればいいかわからず困っています。

Googleスプレッドシートにそのまま貼れる管理表を作ってください。
列項目は「仕入れ日・商品名・仕入れ価格・個数・販売予定価格・実際の販売数・売上金額・利益・残数・備考」
としてください。
最初の3行に花苗のサンプルデータを入れてください。

あわせて、Googleスプレッドシートで使える計算式も入れてください。
- 売上金額 = 販売予定価格 × 実際の販売数
- 利益 = 売上金額 - 仕入れ価格 × 実際の販売数
- 残数 = 個数 - 実際の販売数

出力は、次の2つに分けてください。
1. Googleスプレッドシートに貼り付ける表
2. 計算式の意味を初心者向けに説明したメモ

確認ポイント

  • 列名が自分の業務に合っているか
  • サンプルデータを見て「使えそう」と思えるか
  • 不要な列・欲しい列はないか

4自分の業務に当てはめて「自分版」を作る（10分）

まなんちゃんのプロンプトを見本に、自分の業務で使える管理表やツールを考えて作ります。

自分版プロンプトの作り方

下のテンプレートの【】を自分の言葉で埋めてください。

私は【自分の業種・やっていること】をしています。
今、【何の業務・作業】に困っています（月に約【時間/頻度】かかっています）。

【欲しいもの: 表・リスト・文章テンプレなど】を作ってください。
項目は「【列名や項目名を列挙】」としてください。
【最初のサンプルデータや具体例】も入れてください。

業種別ヒント

業種 何を作ると便利か 欲しい出力の例
ハンドメイド販売 材料費・制作時間の原価計算表 「材料名・単価・使用量・材料費・制作時間・販売価格・利益率」の表
アフィリエイト / ブログ 記事進捗管理シート 「キーワード・タイトル案・文字数目標・進捗・公開日」の表
SNS運用代行 月次報告書テンプレ 「投稿数・いいね合計・フォロワー増減・今月のハイライト・来月の施策」の雛形
副業未定 / 何でもやる系 自分の業務時間の棚卸し 「業務名・月の作業時間・AIで置き換えられるか・優先度」の表
書けない人へ: 「私は【副業の内容】をしています。AIで何を自動化するのがおすすめか3つ教えてください」と聞いてOKです。

5AIの答えをもとに深掘りする（追加質問の練習）（8分）

最初の答えが返ってきたら、次の3つの「深掘り質問」から1つを選んで試します。

「もっと自分の業務に合わせてほしい」場合

私の場合は○○という特徴があります。
それに合わせて表の項目を修正してください。
特に「○○」という列を追加してほしいです。

「別の使い方も試したい」場合

この管理表を使って、月ごとの売上合計と利益率を
自動計算する数式もGoogleスプレッドシート向けに教えてください。

「もっと簡単にしたい」場合

少し複雑すぎます。
初めて使う人でも迷わない、最低限の列だけに絞ったシンプル版を作ってください。
ポイント: 最初から完璧を求めなくていいです。「なんか違う」を言葉にして返すことが練習です。

6グループで見せ合う（2分）

  • 自分が作ったプロンプトとAIの返答をもう1人と見せ合う
  • 「これ使えそう」「ここが難しかった」を一言ずつ

7持ち帰り：次の一歩（1つ選ぶ）

  • 今日作った管理表を実際の業務データで埋めてみる
  • 今日作ったプロンプトを保存して、来週の実務で使ってみる
  • 「もっとこうしたい」という改善案を次回のオフ会で共有する

8行き詰まったときのチートシート

困ったこと やること
質問の型が思い浮かばない 「私は○○しています。AIで効率化できそうな業務を3つ挙げてください」と聞く
返ってきた答えが使えない 「もっと私の業務に具体的に合わせてください」と返す
何を作ればいいかわからない 「私の業務で毎回時間がかかることは何ですか？ヒントをください」と書く
うまくいかなくて止まっている 「ここまで試しました。次に何を直せばいいですか」と Claude に聞く

9グループワーク終了後に全員で確認すること

  • 今日試したプロンプトをスマートフォンのメモに保存する
  • 次回のオフ会までに「1回だけ自分の業務でAIを使う」を宿題にする
  • 困ったことは LINEグループで気軽に相談OK
